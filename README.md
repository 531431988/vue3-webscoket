# vue antd 练习

---

### 使用说明

- :point_right: 安装：npm i 或 cnpm i 或 yarn add
- :point_right: 运行：npm start 或 yarn start
- :point_right: 打包：npm run build 或 yarn build

### 文件结构

```shell
├── components      组件库
├── public          打包所需静态资源
├── template        组件开发模版文件
└── src
    ├── api         AJAX请求
    └── assets      项目静态资源
        ├── icons   自定义图标资源
        └── images  图片资源
    ├── components  业务组件
    ├── mock        mock模拟数据
    ├── router      路由配置
    ├── views       页面文件
```
